# LazerGrabberV1.0

# Help: This version of the grabber only supports **Windows**.

# Features
- Grabs Roblox cookies.
- Grabs Minecraft sessions
- Grabs Google Chrome passwords
- Grabs Discord token
- Grabs victim machine info
- Grabs IP address, geolocation
